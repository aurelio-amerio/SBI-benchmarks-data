# SBI-benchmarks
This repository contains a series of benchmarks for simulation based inference and tools to make a huggingface dataset.
It is used to create the data for the [SBI-benchmarks dataset on Huggingface](https://huggingface.co/datasets/aurelio-amerio/SBI-benchmarks).


To install this package, use:
```bash
pip install git+https://github.com/aurelio-amerio/SBI-benchmarks-data.git --extra-index-url https://download.pytorch.org/whl/cpu

```

See `make_dataset.py` for the basic functionality.
